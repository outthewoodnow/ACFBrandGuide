# Athlete Connections Foundation Brand Guide Site

Static, GitHub Pages-ready brand guide for Athlete Connections Foundation.

## Brand Tokens

- Main Color: `#006BA0`
- Secondary Color: `#F37C31`
- Font: `Source Sans 3`

## Files

- `/index.html` - site markup
- `/styles.css` - brand guide styling
- `/script.js` - copy/download interactions
- `/assets/acf-logo-primary.png` - primary logo asset (add this file)

## Setup

1. Place your logo at:
   - `assets/acf-logo-primary.png`
2. Open `index.html` locally.
3. Push to GitHub.

## GitHub Pages Deploy

1. Push repository to GitHub.
2. Go to **Settings → Pages**.
3. Source: **Deploy from a branch**.
4. Branch: `main`, folder: `/ (root)`.
5. Save.

## Notes

- The site includes copy-to-clipboard buttons for colors and snippets.
- Download buttons generate CSS/JSON/TXT brand token files client-side.
- Typography uses Google Fonts (`Source Sans 3`).
